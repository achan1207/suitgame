Description Game suit

Hallo selamat data di web kami, di sini kami membuat sebuah game generasi 3 di mana game kita ini adalah game suit gunting, batu, kertas. Permainan gunting batu kertas (atau dikenal sebagai "rock-paper-scissors" dalam bahasa Inggris) adalah permainan yang telah ada selama berabad-abad dan dianggap sebagai permainan sederhana yang dimainkan oleh dua orang. Setiap pemain memilih satu dari tiga simbol—gunting, batu, atau kertas—dan hasilnya ditentukan berdasarkan aturan yang sederhana:

  • Gunting mengalahkan kertas (karena gunting memotong kertas),
  • Kertas mengalahkan batu (karena kertas membungkus batu),
  • Batu mengalahkan gunting (karena batu menghancurkan gunting).

    • Description Game tetris

    Tetris adalah salah satu permainan video paling terkenal di dunia, yang pertama kali dibuat oleh Alexey Pajitnov pada tahun 1984 di Moskow, Uni Soviet (sekarang Rusia). Permainan ini tidak hanya menjadi salah satu ikon budaya pop, tetapi juga dikenal sebagai pelopor dalam dunia permainan puzzle.
